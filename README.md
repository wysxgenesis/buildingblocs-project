# Functions:
* users can use indicate their symptoms in the symptoms page to see if it is mild or severe and can then decide if they should visit a doctor. they can also indicate their diagnosis and medication after visiting the doctor
* users can indicate the type of medicine, dosage and time to take it in the reminders page and the web will send a reminder at the time indicated

# How to set up: 
* ensure that you have react downloaded, go to terminal, `cd my-app` then `npm run dev`
